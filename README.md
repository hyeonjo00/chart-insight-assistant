## Quick Summary

- Next.js App Router with TypeScript and Tailwind CSS
- OpenAI-powered chart analysis API route
- Drag-and-drop image upload with preview and validation
- Local analysis history stored in the browser
- AdSense-ready global script, ad banner component, and `ads.txt`
